한국어
## 주요 기능

- **근무시간 기록**
  - 일일 출퇴근 시간 기록
  - 휴게시간, 연장근무, 근무 내역 기록(메모) 기능

- **통계 및 분석**
  - 주간/월간 근무 시간 통계 제공
  - 근무 패턴 시각화 (그래프, 차트)

- **보고서 생성**
  - 회사 제출용 근무 보고서 자동 생성 (PDF/엑셀) (개발중)
  - 근무기록표 수동 생성가능 (PDF) (개발중)

- **사용 편의성**
  - 직관적인 UI로 쉽게 기록 및 확인 가능
  - 로그인/사용자별 데이터 관리

## 기술 스택

- **Backend:** Spring  
- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Database:** MySQL  
- **배포:** Docker, GitHub Actions, AWS EC2


日本語
# WorkTrack

WorkTrackは、勤務時間の記録と統計を簡単かつ便利に管理できるウェブアプリケーションです。  
また、会社提出用の勤務報告書を自動生成することができ、業務効率の向上に役立ちます。

## 主な機能

- **勤務時間の記録**
  - 日々の出勤・退勤時間の記録
  - 休憩時間、残業時間、勤務内容のメモ機能

- **統計と分析**
  - 週次/月次の勤務時間統計の提供
  - 勤務パターンの可視化（グラフ、チャート）

- **報告書の作成**
  - 会社提出用の勤務報告書の自動生成（PDF/Excel）（開発中）
  - 勤務記録表の手動作成可能（PDF）（開発中）

- **使いやすさ**
  - 直感的なUIで簡単に記録と確認が可能
  - ログイン/ユーザー別のデータ管理

## 技術スタック

- **バックエンド:** Spring  
- **フロントエンド:** HTML, CSS, Vanilla JavaScript  
- **データベース:** MySQL  
- **デプロイ:** Docker, GitHub Actions, AWS EC2
